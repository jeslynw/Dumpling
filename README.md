# SmartNotebook

-- Frontend --
1. Install Next.js ```npx create-next-app frontend```
2. Inside frontend folder -> Add tiptap: ```npx @tiptap/cli@latest add simple-editor```

-- Create env --
Create env: ```conda create -p ./env python=3.13 -y```
Activate env: ```conda activate ./env```
Install dependencies: ```pip install -r requirements.txt``
